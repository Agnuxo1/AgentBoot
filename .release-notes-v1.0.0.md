## AgentBoot v1.0.0 — Production-ready bare-metal installer

AgentBoot turns any phone or laptop into a portable sysadmin.
Connect to a server with no OS, no keyboard, no monitor — chat with the
on-device AI, and it identifies the hardware, recommends the right OS,
downloads and verifies the installer, flashes a USB stick, generates
the autoinstall config, and drives the install through a resumable
state machine.

### What's in v1.0.0

- **Hardware detection** — local (`psutil`, `/proc`, `wmic`, `lspci`) and remote over SSH.
- **OS recommender** — 13-OS curated catalogue with architecture / RAM / disk / NIC gating.
- **ISO catalogue + downloader** — resumable HTTP `Range`, SHA256 verification against vendor `SHA256SUMS` (GNU / BSD / FreeBSD formats).
- **USB flasher** — cross-platform enumeration (Linux lsblk, Windows WMI, macOS diskutil) + dd-style writer with three-layer safety (removable-only, not-system-disk, confirm-token match).
- **Autoinstall generators** — cloud-init (subiquity), Debian preseed, Red Hat kickstart, Windows unattend.xml.
- **Bare-metal collector** — stdlib-only script for minimal live OSes. JSON-over-serial protocol with `hw.report`, `ping`, `config.write`, `system.reboot`, `system.poweroff`.
- **Orchestrator** — 10-state install session (`INIT → DETECTING → RECOMMENDING → DOWNLOADING → FLASHING → CONFIGURING → INSTALLING → VERIFIED → DONE` + `FAILED`) with atomic JSON persistence and `--resume`.
- **LLM router** — on-device Qwen3.5 0.8B (llama.cpp) with Claude / Gemini fallback; priority-ordered, `LLMUnavailable` skips, `LLMError` fails hard.
- **Professional CLI** — 10 subcommands (`chat`, `detect`, `recommend`, `list-isos`, `download`, `list-devices`, `flash`, `gen-config`, `session`, `install`) plus `--config` / `-v`.
- **Quality plumbing** — unified `AgentBootError` hierarchy, JSON config with XDG / APPDATA resolution, idempotent logging setup.
- **Docs** — [USAGE](docs/USAGE.md), [OPERATOR](docs/OPERATOR.md), [DEVELOPER](docs/DEVELOPER.md), [COLLECTOR](docs/COLLECTOR.md).

### Tests

220 passed, 1 skipped. `test_local_llm.py` is excluded in CI (requires a real GGUF model and a working `llama-cpp-python` binding).

### Install

```bash
pip install agentboot            # core
pip install agentboot[cloud]     # + Anthropic / Gemini
pip install agentboot[local]     # + llama-cpp-python
pip install agentboot[all]       # everything
```

### Design principle

No fake features, no stubs, no half-finished implementations. Every
exported symbol has a real implementation and at least one passing
test. Failure is always explicit (`AgentBootError`), side-effects
are always behind confirm-token safety rails, and every destructive
action is idempotent once the session records it.
